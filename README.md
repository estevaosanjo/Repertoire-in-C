Repositório dedicado a projetos simples de estudo em Linguagem C, criado com o objetivo de praticar e consolidar conhecimentos em lógica de programação, estruturas de dados, algoritmos de ordenação, entre outros conceitos essenciais. 
 - Functions: Recursão, Pesquisa Binária, Estrutura de dados - junção de todas as funções que tem dentro do sistema de controle e em projetos de estudos. Essas funções são para teste, e posteriormente serão adicionadas em seu projeto principal.
 - Projects: Projetos criados para estudo e manipulação de estruturas básicas 
 - Sistema: Sistema de controle (Cadastro, login, validação)
